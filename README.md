## How It Was Built

HART was built by one developer using **Claude Code with Claude Sonnet** as the primary coding engine — every edge function, database schema, frontend component, and Solana wallet integration was built in collaboration with Claude Code in the terminal.

**Claude Opus 4.7** was brought in for higher-level reasoning: reviewing architecture documents, adding inline documentation, catching bugs, and planning the next phase of the product.

This isn't a demo. It's a live, working platform — built lean, built fast, built with Claude.

## What Winning Unlocks

The $500 in credits would go directly into shipping the next phase of HART using **Claude Opus 4.7** end to end:

- **Direct fiat payments** — buyers pay with card, no crypto wallet required
- **Community Collage System** — collaborative art pieces built by multiple artists on the platform
- **3D Art Environment** — a spatial gallery where collectors can walk through and experience art in 3D
- **Artist Expansion Tools** — full Supabase-powered artist onboarding, portfolio management, and analytics

These aren't stretch goals. They're the next sprint. The credits make the difference between planning them and shipping them.
